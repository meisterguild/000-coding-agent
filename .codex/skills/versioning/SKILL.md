# Versioning

## Overview

ファイルの変更に関するルール
人間の承認が通ったファイル変更についてはgit commitすること

## Commit message
- 対応中のタスクがある場合は、そのタスクのタイトルをコミットメッセージにすること
- 対応中のタスクがない場合は、日本語のコミットメッセージを付けること

## Prerequisites

.gitディレクトリがプロジェクトルートになければ、git initを実行する
